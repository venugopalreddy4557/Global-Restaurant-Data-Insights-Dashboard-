1. Project Title
Global Restaurant Data Insights Dashboard

2. Project Overview

This Power BI project analyzes a global restaurant dataset to uncover insights about cuisine preferences, restaurant distribution across major cities, country-wise presence, pricing patterns, and the impact of online delivery on restaurant ratings.
The goal of this project is to demonstrate practical data visualization skills, data cleaning, and storytelling using Power BI.
